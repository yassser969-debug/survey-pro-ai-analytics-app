# Pro AI Survey Analytics Dashboard

A Streamlit web app for lecturer and student questionnaire analysis.

## Run locally

```bash
pip3 install -r requirements.txt
python3 -m streamlit run app.py --server.port 8503
```

## Features

- Upload lecturer and student Excel files
- Executive overview
- Data quality audit
- Single-question analysis
- Filtered analysis
- Crosstab analysis
- Likert reliability analysis with Cronbach's Alpha
- Text-response analysis
- AI interpretation and AI report generation
- Export Excel report
